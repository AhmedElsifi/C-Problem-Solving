## 🚀 How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/cpp-problem-solving.git```

2. Navigate to the project folder:

   ```bash
   cd cpp-problem-solving
   ```
3. Compile and run a file (example):

   ```bash
   g++ Basics/loops.cpp -o loops
   ./loops
   ```
